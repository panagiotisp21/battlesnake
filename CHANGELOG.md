# Battlesnake Project Changelog

This document tracks the evolution of the Battlesnake project from initial setup to the current production release.

## [v1.0.0 (The Survival Milestone)] - 2026-05-15

**Category:** Major Release / Iteration 3

Implementation of Flood Fill algorithm (BFS) to avoid dead ends. Achieved >80% test coverage milestone. Updated author metadata to full team: Alexios Kalmpouros, Panagiotis Peppas, Albert Jefferson Abuy, Ydnar Nick Rico. Refactored server entry points to support TDD.

---

## [v0.3.0 (TDD & Logic Refinement)] - 2026-05-01

**Category:** Iteration 3 Beta

Setup Jest with ES Module support using cross-env and NODE_OPTIONS. Established 'integration.test.js' for API lifecycle validation. Migrated logic to exported functions in index.js.

---

## [v0.2.0 (The Strategic Foundation)] - 2026-04-15

**Category:** Iteration 2

Implemented Manhattan Distance calculation for food seeking. Added complex collision avoidance for other snakes. Refined move logic to prioritize safety over random selection.

---

## [v0.1.1 (Metadata Update)] - 2026-03-30

**Category:** Patch

Updated snake appearance: set head to 'beluga', tail to 'curled', and primary color to #45b345ad.

---

## [v0.1.0 (The Initial Prototype)] - 2026-03-15

**Category:** Iteration 1

Initial project setup with basic Express server. Implemented fundamental boundary checks (avoid walls) and self-collision prevention (avoid neck).

---

## [v0.0.1 (Genesis)] - 2026-03-01

**Category:** Pre-Release

Forked starter repository. Configured basic package.json scripts for start/test. Verified connectivity with Battlesnake engine.

---
